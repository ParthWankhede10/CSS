System Fonts & Fallbacks

This page shows how font-family, font-weight, font-style, and font-variant behave across headings, paragraphs, and code blocks.

Serif vs Sans-Serif vs Monospace

Serif paragraph — typically used for long-form reading. Notice the small “feet” on letter strokes.

Sans-serif paragraph — great for UI and screens. Clean, no extra stroke endings.

Monospace paragraph — every character has equal width; ideal for code and tabular alignment. 

// Monospace shines for code:

  return `Hello, ${name}!`;
}
