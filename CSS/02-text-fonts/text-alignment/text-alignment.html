Text Alignment Examples

We’ll demo left, right, center, justify, and text-align-last.

Left (default)

Left-aligned text is the default for LTR languages. It’s easy to read and natural for most readers.

Right

Right alignment is sometimes used for dates or small UI elements to visually balance a layout.

Center

Center alignment is popular for titles or short lines of text, but avoid using it for long paragraphs.

Justify (+ Last Line)

Justified text spreads space so each line touches both left and right edges. On the last line, we can override the alignment using text-align-last to avoid awkward spacing artifacts on a short final line.

Demonstrating text-align-last

This paragraph is justified for all but the final line. The last line is centered via text-align-last: center to reduce odd word gaps.
